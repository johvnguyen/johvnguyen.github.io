Darkspace Internet Traffic Data Analysis

(Python, FastAPI, Pandas, HTML/CSS, Javascript, d3.v5, Docker, docker-compose)

Source code:

In this project, my team and I analyze darkspace internet traffic and attempt to distinguish malicious behaviors using big data tecniques such as t-SNE and UMAP. We create a dashboard with a frontend and backend powered by FastAPI in order to visualize high dimensional vectors of network statistics.

This project was developed by a team of five students as part of our CSE 6242: Data and Visual Analytics class during the Fall 2024 semester. My contributions include parsing features from PCAP data and designing the frontend. For perosnal study, I later containerized our dashboard using Docker and docker-compose.
